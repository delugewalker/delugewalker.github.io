本文介绍几种跨平台BitTorrent客户端，并详细说明其在Linux操作系统上的使用方法。

1 Deluge

1.1 简介

Deluge是一款跨平台BitTorrent客户端，支持Linux (Debian, Fedora, OpenSUSE, Arch, Gentoo)、Win、Mac、FreeBSD多个平台，非常稳定，而且支持单一种子限速功能。部分PT站会对单一种子的速度有限制，如果超速，账号可能会被警告甚至封禁。如果PT站有限速要求，推荐使用Deluge。比较遗憾的是，Deluge没有官方的CentOS源。如果需要在CentOS安装，可尝试下第三方的源。各版本的安装包均在Deluge官网可以下载。

以下说明Ubuntu安装Deluge的具体步骤。

1.2 Ubuntu安装Deluge

  • 添加Deluge PPA Repository以防止无法安装最新版本

    sudo add-apt-repository ppa:deluge-team/stable
sudo apt update
sudo apt install deluge

  • 安装控制台版本Deluge(推荐，可使用web打开并在后台运行)

    sudo apt-get install deluged deluge-web deluge-console

  • 安装可视化版本Deluge

    sudo apt-get install deluge

1.3 Deluge使用

  • deluge-web使用

    • 启动deluge-web

      deluge-web --fork

      --fork表示在后台运行

    • 浏览器打开可视化界面
      在本地浏览器输入127.0.0.1:8112即可访问deluge-web，默认密码：deluge

    • 远程ipv6地址访问
      deluge-web的端口只对ipv4监听，我们可以考虑使用socat进行监听端口转发，将内网ipv4端口转发至公网ipv6端口，从而达到外网访问的目的。

      • 安装socat
      sudo apt install socat
      • 监听端口转发
      socat TCP6-LISTEN:8113,reuseaddr,fork TCP4:10.31.13.227:8112

      ipv6端口选择8113是因为8112端口已被deluge-web进程占用。

  • deluge-dkg使用

    deluge

2 qBittorrent

2.1 简介

qBittorrent项目旨在为μTorrent提供一种开源软件替代品。此外，qBittorrent在所有主要平台（FreeBSD，Linux，macOS，OS/2，Windows）上运行并提供相同的功能。qBittorrent基于Qt工具包和libtorrent-rasterbar库。

以下说明Ubuntu安装qBittorrent的具体步骤。

2.2 Ubuntu安装qBittorrent

  • 添加qBittorrent PPA Repository以防止无法安装最新版本

    sudo add-apt-repository ppa:qbittorrent-team/qbittorrent-stable
sudo apt update

  • 安装qBittorrent(WebUI)

    sudo apt-get install qbittorrent qbittorrent-nox

2.3 Deluge使用

qbittorrent-nox --webui-port=8114

此进程会占用当前命令行，建议在tmux/screen中打开。
